#   FaidChatbot
-A django project that provides first aid guidance and ussd chatbot interface through africa's talking

## Its features
-Provides step by step first aid instruction with a maximum of upto 5 steps
-Has hardcoded instructions and uses gemini api for unknown queries
-USSD integration for easy mobile access
-Handles common emmergencies like burns,CPR,Fainting etc
-Has authentication segment that allows user to login or signup and recognizes user after loging in
